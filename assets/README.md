# Assets

This directory contains project assets like logos, screenshots, and diagrams.

## Planned Assets

- `logo.png` - Vertex CLI logo
- `screenshot-menu.png` - Screenshot of interactive menu
- `screenshot-chat.png` - Screenshot of chat interface
- `architecture.svg` - Architecture diagram
- `demo.gif` - Animated demo

## Contributing Assets

When adding assets:
- Use PNG for screenshots (compressed)
- Use SVG for diagrams (when possible)
- Keep file sizes reasonable (<500KB for images)
- Use descriptive filenames
